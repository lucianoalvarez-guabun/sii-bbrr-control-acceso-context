# Registro Global de Historias de Usuario (HdU)

## Objetivo
Mantener un registro centralizado de todas las HdU del proyecto con IDs únicos para evitar duplicados y facilitar trazabilidad.

## Nomenclatura
- **Formato:** `HdU-XXX-Nombre-Funcionalidad.md`
- **ID:** Número secuencial de 3 dígitos (001-999)
- **Nombre:** Descripción corta en formato kebab-case
